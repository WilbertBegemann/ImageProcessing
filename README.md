# What do you need to run
- python 3
- pip install PILLOW
- pip install numpy

# How to run
```
    python3 gaussian.py
```
or
```
    python3 sobel.py
```
or
```
    python3 atan2sobel.py
```
# estimated run time
+-4 seconds with Black_Circle.jpg
